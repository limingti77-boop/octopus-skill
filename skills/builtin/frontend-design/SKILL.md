# frontend-design

## 名称
frontend-design

## 描述
创建美观、高质量的前端界面，支持 React、Vue、HTML/CSS 等多种技术栈。

## 触发条件
- 用户请求创建网页、界面、UI 组件
- 用户提到"前端"、"页面"、"界面"、"UI"、"网页设计"
- 用户需要构建 Landing Page、Dashboard、表单等前端项目
- 用户需要美化或重构现有前端代码

## 能力说明
- 创建响应式布局和现代化 UI 设计
- 支持 React、Vue、Next.js、HTML/CSS 等技术栈
- 遵循最佳实践和设计原则
- 生成整洁、可维护的前端代码
- 支持主题定制和样式优化
- 集成 Tailwind CSS、shadcn/ui 等流行工具
